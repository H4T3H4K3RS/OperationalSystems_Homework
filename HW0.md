# Примеры современных ОС, рекомендуется в ВС общего назначения

## 1. Операционные системы для мейнфреймов:

* `z/OS`: операционная система для мейнфреймов, разработанная `IBM` для использования на мейнфреймах `IBM`. Он известен
  своей масштабируемостью и безопасность.
* `VSE/ESA`: операционная система для мейнфреймов, разработанная `IBM` для использования на мейнфреймах `IBM`. Он
  известен своей надежностью и простота использования.
* `BS2000/OSD`: операционная система для мэйнфреймов, разработанная `Fujitsu` для использования на мейнфреймах Fujitsu.
  Он известен своим масштабируемость и безопасность.

## 16. Операционные системы гипервизора

* `VMware ESXi`: Гипервизор на «голом железе», который работает непосредственно на оборудовании хоста для управления
  виртуальными машинами.
* `Microsoft Hyper-V`: гипервизор типа 1, разработанный `Microsoft` для `Windows`, он позволяет создавать и управлять
  виртуальными машины
* `Xen`: гипервизор типа 1 с открытым исходным кодом, который широко используется в корпоративных средах для создания и
  управления виртуальными машины.
